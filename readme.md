# Awesome Tool - 实用工具集合

一个基于 Next.js 构建的实用工具集合网站，提供各种便利的在线工具。

## 🚀 功能特性

- 🖼️ **图片压缩**：支持 JPEG、PNG、WebP 格式的图片压缩
- 📱 **响应式设计**：适配桌面端和移动端
- 🎨 **现代化 UI**：使用 Tailwind CSS 构建的美观界面
- ⚡ **高性能**：基于 Next.js 的服务端渲染

## 📦 安装

1. 克隆项目
```bash
git clone https://github.com/your-username/awesome-tool.git
cd awesome-tool
```

2. 安装依赖
```bash
npm install
```

3. 启动开发服务器
```bash
npm run dev
```

4. 打开浏览器访问 [http://localhost:3000](http://localhost:3000)

## 🛠️ 技术栈

- **框架**: Next.js 14
- **语言**: TypeScript
- **样式**: Tailwind CSS
- **图标**: Lucide React
- **构建**: SWC

## 📱 功能说明

### 图片压缩工具

- 支持批量上传图片
- 可调节压缩质量（10% - 100%）
- 支持输出格式选择（WebP、JPEG、PNG）
- 实时预览压缩效果
- 显示压缩比例和文件大小对比
- 一键下载压缩后的图片

## 🔧 部署

### 构建生产版本

```bash
npm run build
npm start
```

### 部署到 Vercel

1. 推送代码到 GitHub
2. 在 Vercel 中导入项目
3. 自动部署完成
